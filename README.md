# Loopix

Увлекательная головоломка, в которой нужно соединять блоки и создавать замкнутые цепи.

## 🎮 Как играть
- Выбирайте блоки из нижней панели
- Вращайте их свайпом или касанием
- Размещайте на поле, создавая замкнутые петли
- Чем больше петель — тем больше очков!

## 🛠 Технологии
- HTML5 Canvas
- Cordova для Android сборки
